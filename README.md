# Usage
Install **Share to Delete**.

* From Google Play: Coming soon...
* Manual download: https://github.com/pingpongboss/Share-To-Delete/downloads

Take a picture. **Share to Delete** automatically appears.

![Turbo Mode - Camera Overlay](https://github.com/pingpongboss/Share-To-Delete/raw/master/web/images/camera%20overlay.png)

Click to delete the picture.
